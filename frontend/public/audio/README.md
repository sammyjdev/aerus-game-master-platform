# Audio Assets

Place here the audio files referenced by `src/constants/audio.ts`.

Expected structure:

```text
audio/
├── idle/
│   ├── idle_01.mp3
│   ├── idle_02.mp3
│   ├── idle_03.mp3
│   └── idle_04.mp3
├── blunt_hit.mp3
├── bow_shoot.mp3
├── sword_hit.mp3
└── unarmed_hit.mp3
```

For the MVP, you can use a single `placeholder.mp3` and copy/rename it to the filenames above.
